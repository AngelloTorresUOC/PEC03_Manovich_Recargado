# PEC03_Manovich_Recargado
Trabajo presentado por Angello Torres De la torre, para la asignatura de Cultura Digital, actividad 03 “Visionando el futuro con las gafas de Manovich: Redescubriendo la hibridación”.  En este texto se plantean dos ejemplos modernos sobre la hibridación de medios, usando el criterio y los planteamientos de Lev Manivich.
